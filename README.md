#  Asteroid Dodger Game — LPC2138 | 16x4 LCD

A simple embedded game built using the **LPC2138 ARM7 microcontroller**, where the player controls a spaceship to dodge falling asteroids. The score increases with every successful dodge, and gameplay ends on collision.



##  Game Overview
- The spaceship moves **left or right** using input buttons.
- **Asteroids fall** from the top of the screen on a **16x4 LCD display**.
- **Game ends** when the asteroid hits the spaceship.
- **Score is displayed** in real-time.
- Developed for **embedded systems practice & LCD graphics handling**.



##  Features
- Real-time spaceship control with push buttons  
- Random asteroid spawning logic  
- Clear game graphics using custom LCD characters  
- Score tracking mechanism  
- Collision detection  



##  Hardware / Components Used
- **LPC2138 ARM7 Microcontroller** – Main system controller  
- **16x4 Character LCD Display** – For rendering game elements  
- **Push Buttons** – Left & right movement  
- **5V Power Supply** – System powering  



##  Software Tools Used
- **Keil uVision 4**  
- **Proteus 8 Professional** *(for simulation)*  



##  How to Play
- Use **Left** and **Right** buttons to move the spaceship  
- **Avoid falling asteroids** from the top  
- **Score increases** for every successful dodge  
- **Game over** if an asteroid hits the spaceship  
- Press **Reset** to restart
